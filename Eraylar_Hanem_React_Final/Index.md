# 🏰 Eraylar Hanem React Final - Second Brain

Bu vault, Eraylar Hanem uygulamasının tüm teknik mimarisini, modül detaylarını, buton işlevlerini ve veri bağlarını içeren güncel dokümantasyon merkezidir.

## 📱 Uygulama Genel Bakış
Eraylar Hanem, Görkem ve Esra için geliştirilmiş, her şeyi kapsayan bir aile yönetim merkezidir.

### 🛠 Teknoloji Yığını
- **Frontend:** React + Vite
- **State Yönetimi:** Zustand (Persist & Supabase Sync)
- **Veritabanı:** Supabase (PostgreSQL + Realtime)
- **Styling:** CSS Variables + Glassmorphism
- **İkonlar:** Lucide React
